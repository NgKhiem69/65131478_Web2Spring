# 💻 BÀI 2 - JSP BASIC & LOGIN DEMO

## 📌 Mô tả
Ứng dụng web đơn giản sử dụng **JSP (Java Server Pages)** để thực hành các thành phần cơ bản của JSP và xử lý đăng nhập.

Chức năng:
- Hiển thị nội dung động bằng **JSP Expression**
- Sử dụng **JSP Scriptlet và Declaration**
- Tạo **form đăng nhập**
- Xử lý dữ liệu đăng nhập bằng **JSP**
- Điều hướng trang bằng `response.sendRedirect()`
- Hiển thị trang **User Profile** khi đăng nhập thành công

---

## 🚀 Công nghệ sử dụng
- JSP (Java Server Pages)
- Apache Tomcat 10.1
- HTML, CSS
- Java

---

## ▶️ Cách chạy chương trình

1. Import project vào Eclipse  
2. Cấu hình Apache Tomcat  
3. Chạy project trên server  
4. Truy cập:

http://localhost:8080/project/Login.html

---

## 📝 Cách sử dụng

- Mở trang **Login**
- Nhập thông tin đăng nhập
- Nhấn **Login**

Thông tin đăng nhập mẫu:

Username: ABC  
Password: MNK  

Kết quả:

- Đăng nhập đúng → chuyển đến **User Profile**
- Đăng nhập sai → quay lại **Login**
